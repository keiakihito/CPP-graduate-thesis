 1. Landing Page – Album Catalog
      - Visitor sees a grid/list of available albums on the first page.
      - Each album entry is clickable.
  2. Album Detail
      - Selecting an album shows its metadata (title, artist, year) and the track list.
      - Every listed track provides a “View” action.
  3. Track Preview
      - Selecting a track opens the waveform view.
      - User can play/pause from the current selection, jump ±5s, and resume at the last playhead.
  4. Segment Selection
      - User can drag/stretch/shrink a selection region on the waveform.
      - System shows the current range and playback stays within that region.
  5. Clip Download
      - When the user confirms the range, the backend trims the audio accordingly.
      - A download is initiated so the clip saves to the visitor’s device.
